App Store Release & Review Benchmark

Compare releases, ratings, and review samples around app release windows.

For Mobile product and ASO teams

Public sources: Public Apple App Store and Google Play metadata

Starter cap

Open on ApifyStarter JSONFull-report JSON

Decisions this report supports

  • Compare rating and review signals around release windows
  • Identify post-release regressions that need product review
  • Benchmark a small app set before generating a category report

Data limitation: Storefront ratings and review availability vary by country and time and are not a complete measure of product quality.

Buyer-capped starter input

The maintained starter is expected to deliver one `app-benchmarked` event at $0.015 and one `app-review-regression` event at $0.10 when the sample evidence still matches. The buyer-controlled cap remains $0.50.

Run with the Apify CLI

After cloning the starter repository and signing in with the Apify CLI, run the maintained capped input directly:

apify actors call taroyamada/app-release-category-review-benchmark-report --input-file inputs/app-release-category-review-benchmark-report.json --output-dataset
